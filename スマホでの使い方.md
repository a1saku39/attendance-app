# スマホでWebアプリを使う方法

スマホでこのWebアプリを使うには、HTMLファイルをインターネット上に公開する必要があります。
以下、おすすめの方法を3つご紹介します。

## 方法1: GitHub Pages（無料・おすすめ）

### 手順
1. **GitHubアカウントを作成**（まだの場合）
   - https://github.com にアクセスして無料アカウントを作成

2. **新しいリポジトリを作成**
   - 「New repository」をクリック
   - リポジトリ名: `attendance-app`（任意）
   - 「Public」を選択
   - 「Create repository」をクリック

3. **ファイルをアップロード**
   - 「uploading an existing file」をクリック
   - 以下のファイルをドラッグ&ドロップ:
     - `index.html`
     - `style.css`
     - `script.js`
   - 「Commit changes」をクリック

4. **GitHub Pagesを有効化**
   - リポジトリの「Settings」タブをクリック
   - 左メニューから「Pages」を選択
   - 「Source」で「main」ブランチを選択
   - 「Save」をクリック

5. **URLを取得**
   - 数分待つと、以下のようなURLが表示されます:
   - `https://ユーザー名.github.io/attendance-app/`
   - このURLをスマホのブラウザで開けばOK！

### メリット
- 完全無料
- 更新も簡単（ファイルを再アップロードするだけ）
- 安定している

---

## 方法2: Netlify（無料・最も簡単）

### 手順
1. **Netlifyにアクセス**
   - https://www.netlify.com にアクセス
   - 「Sign up」で無料アカウントを作成

2. **ドラッグ&ドロップでデプロイ**
   - ログイン後、「Add new site」→「Deploy manually」をクリック
   - `AttendanceApp` フォルダ全体をドラッグ&ドロップ
   - 自動的にデプロイされます

3. **URLを取得**
   - `https://ランダムな名前.netlify.app` のようなURLが発行されます
   - このURLをスマホで開けばOK！

### メリット
- **最も簡単**（ドラッグ&ドロップだけ）
- 完全無料
- 自動的にHTTPSになる

---

## 方法3: Vercel（無料・高速）

### 手順
1. **Vercelにアクセス**
   - https://vercel.com にアクセス
   - 「Sign up」で無料アカウントを作成

2. **プロジェクトをデプロイ**
   - 「Add New」→「Project」をクリック
   - GitHubと連携するか、ファイルを直接アップロード
   - 自動的にデプロイされます

3. **URLを取得**
   - `https://プロジェクト名.vercel.app` のようなURLが発行されます

### メリット
- 高速
- 完全無料
- 自動デプロイ機能が便利

---

## 方法4: ローカルネットワークで試す（一時的なテスト用）

同じWi-Fiに接続されたスマホで試すだけなら、この方法が最も簡単です。

### 手順
1. **PCのIPアドレスを確認**
   - コマンドプロンプトで `ipconfig` を実行
   - 「IPv4 アドレス」をメモ（例: 192.168.1.5）

2. **簡易サーバーを起動**
   - AttendanceAppフォルダで以下のコマンドを実行:
   ```
   python -m http.server 8000
   ```
   または
   ```
   npx http-server -p 8000
   ```

3. **スマホでアクセス**
   - スマホのブラウザで以下にアクセス:
   - `http://192.168.1.5:8000`（IPアドレスは自分のものに置き換え）

### 注意点
- PCとスマホが**同じWi-Fi**に接続されている必要があります
- PCの電源を切るとアクセスできなくなります
- あくまで一時的なテスト用です

---

## おすすめの方法

### 初めての方 → **Netlify（方法2）**
- 最も簡単で、ドラッグ&ドロップだけで完了
- 更新も簡単

### 長期的に使う → **GitHub Pages（方法1）**
- バージョン管理もできる
- 無料で安定している

### 一時的なテスト → **ローカルネットワーク（方法4）**
- 公開せずに試せる
- 設定が最も簡単

---

## 公開後の設定

どの方法でも、公開後は以下の設定を行ってください:

1. **スマホでURLにアクセス**
2. **画面下部の「設定（管理者用）」を開く**
3. **GAS WebアプリのURLを入力**
4. **「保存」をクリック**

これで、スマホから打刻できるようになります！

---

## セキュリティについて

- このアプリは認証機能がないため、URLを知っている人は誰でもアクセスできます
- 社内のみで使う場合は、URLを社員にのみ共有してください
- より高度なセキュリティが必要な場合は、認証機能の追加をご検討ください
